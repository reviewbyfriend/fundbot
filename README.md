# FundBot v2.1 LINE First Admin

แก้ระบบอนุมัติจาก LINE ให้ไม่ต้อง login ซ้ำ:

- การ์ดรอตรวจส่งเฉพาะแชตส่วนตัวแอดมินที่ลงทะเบียน LINE แล้ว
- ปุ่ม อนุมัติ / ไม่ผ่าน / ดูหลักฐาน ใช้ signed URL เฉพาะแอดมินคนนั้น
- เปิดจาก LINE ได้เลย ไม่ขึ้น Forbidden
- ไม่ส่งการ์ดสถานะเข้ากลุ่มเมื่อมีคนจ่าย/อัปสลิป/เงินสด/อนุมัติ/Reject
- รองรับ trigger `@fundbkc1` และ `@fundbot`
- คำสั่ง `@fundbkc1 รอตรวจ` จะส่งการ์ดอนุมัติให้แอดมินส่วนตัว

วิธีใช้แอดมิน:

1. เพิ่มแอดมินใน `/admin/admins`
2. แอดมินทักบอทส่วนตัว แล้วพิมพ์:
   `@fundbkc1 สมัครแอดมิน ADMIN_CODE`
3. เมื่อมีสลิปรอตรวจ บอทจะเด้งเข้าแชตส่วนตัวของแอดมิน
4. กดอนุมัติได้ทันทีจาก LINE
